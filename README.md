## 📊 Sales Performance Dashboard (Excel)

### 🔍 Project Overview
This project analyzes sales performance across regions, categories, and time using Microsoft Excel.

---

### 📌 Key Features
- KPI metrics: Total Sales, Profit, Quantity, Orders  
- Sales analysis by region  
- Monthly sales trend  
- Profit by category  
- Top 5 products analysis  
- Interactive slicers (Region & Category)  

---

### 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers  

---

### 📈 Insights
- East region generated the highest sales  
- Electronics category contributed maximum profit  
- Sales declined from January to February
